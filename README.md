# 个人工作站-记账本-日程本

#### 介绍
因为本人比较穷B，需要一个记账本来管理和查看平时的支出，本来是想伸手党直接拿一个现成的用，无奈找不到符合自己需求的，于是自己做了一个，前后端都是基于 js 所搭建，采用的也是无门槛且没有任何花哨的技术，纯属是为了快速简易的当成一个记事本使用，配合简单的增删改查来使用，虽然还存在很多问题，不过无伤大雅不影响使用，各位有需要可以自取使用，或者帮我升级升级完善一下功能或修复一下 BUG 本人更是跪谢～
